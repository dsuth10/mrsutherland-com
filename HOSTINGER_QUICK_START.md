# Hostinger Quick Start - Upload JavaScript File

## Fastest Method (2 minutes)

1. **Log into hPanel**: https://hpanel.hostinger.com/

2. **Click "File Manager"** (in Files section)

3. **Navigate to**: `public_html/wp-content/uploads/`

4. **Click "Upload"** button → Select `dist/assets/flooding-sorter-app.js`

5. **Done!** ✅

Your file will be at: `https://mrsutherland.net/wp-content/uploads/flooding-sorter-app.js`

The WordPress page is already configured to load it.

---

## Need SFTP? (For Premium plans)

**Quick Setup:**
- Enable SSH: **Advanced** → **SSH Access** → **Enable**
- Port: **65002** (not 21!)
- Protocol: **SFTP** in FileZilla
- Host: FTP IP from SSH Access section
- Password: Same as your FTP password

See `HOSTINGER_UPLOAD_GUIDE.md` for detailed instructions.
